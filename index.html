About King of Open Play

More play. Less waiting.

King of Open Play is a free, offline-first organizer for pickleball open play, skill ladders, scoring, and fixed-team tournaments.

♟

Built for organizers

Register large groups, rotate players fairly, assign courts automatically, and record results without paddle stacking.

◉

Private and offline

Session data stays in this browser. There is no account, cloud database, tracking system, or required internet connection after installation.

◇

Free to use

The application is designed as a static website that can be shared publicly and hosted without a paid server.

How to use King of Open Play

From check-in to the final rally.

Follow this organizer workflow. Each section explains what the app does and when to use each control.

Start event Register Run courts Results Tournament Offline data
01

Build the event

Select New session or the settings icon, then choose the event type:

Open PlayEqual-game rotation for social sessions.
Skill LadderMatches players using accumulated results.
Team TournamentFixed doubles teams and round-robin standings.

Set the number of courts, singles or doubles, Play To value, Win By rule, scoring system, and matchmaking method. The event simulation explains the selected setup before you start.

02

Register participants

Open Players. Paste one player name per line and select Add to rotation. Duplicate names are skipped. For testing, generate 16 to 64 sample players.

In Tournament mode, register a team name and two fixed partners. Existing individual players can be converted into pairs.

03

Assign and manage courts

Select Fill open courts. The app prioritizes rested players with fewer games, builds teams according to the chosen matchmaking mode, shows the VS introduction, and optionally announces names.

  • Use the speaker control to enable or mute announcements.
  • Use Substitute a player if someone must leave early.
  • Use Close court when the organizer's court time ends.
  • Closed courts are ignored until reopened.
04

Record rallies or final results

For complete scoring, tap the team that won each rally. Traditional side-out mode tracks serving team, server number, and side-outs automatically.

When time is limited, select Team A won game or Team B won game. For more accurate skill matching, enter the final score. The app validates Play To, Win By, ties, and deuce scores.

Example: Play To 11, Win By 2 accepts 11-9 or 12-10, but rejects 11-10 and 12-9.
05

Run a team tournament

Register at least three fixed teams, then generate the schedule. The app creates all round-robin matches, prevents one team from being assigned to two courts at once, and automatically fills free courts.

Open Standings to view games played, wins, losses, points for, and point difference. Rankings use wins first, then point difference.

06

Save, export, and work offline

Every change is saved automatically in the current browser. Use Export to download a JSON backup and Import to restore it on another device.

Important: Browser data does not sync automatically. Clearing site data removes the local session, so export important events first.

    How to use the application

    A quick organizer walkthrough before starting your first event.

    01

    Choose an event

    Select Open Play, Skill Ladder, or Team Tournament. Configure courts, scoring, and matchmaking.

    02

    Register everyone

    Add individual players in bulk or register fixed tournament teams.

    03

    Fill the courts

    The app assigns eligible players fairly and announces each matchup when the announcer is enabled.

    04

    Record results

    Track each rally, tap the winning team, or enter a validated final score.

    05

    Keep playing

    Finished courts automatically receive the next eligible matchup. Substitute players or close courts when needed.

    06

    Protect the session

    Everything saves locally. Export a JSON backup before clearing browser data or changing devices.
